# CareerForge

Expo + React Native + TypeScript starter with NativeWind styling and Supabase wiring.

## Structure

- `App.tsx` - App entry point
- `src/components` - Reusable UI pieces
- `src/screens` - Screen-level layouts
- `src/services` - External service clients
- `src/constants` - Shared design values
- `src/types` - Shared TypeScript types
- `src/utils` - Small helper functions

## Setup

1. Copy `.env.example` to `.env`.
2. Fill in your Supabase URL and anon key.
3. Install dependencies.
4. Run `npm start`.
